ClearCost Widget Demo

Interactive demo of the @clearcost/widget embeddable component.

Note: This demo uses a mock proxy endpoint. In production, replace proxyUrl with your server-side proxy that forwards requests to the ClearCost API with your API key.

1. Default theme

Minimal setup with just data-clearcost attributes. Uses the default light theme with system fonts. 

<div data-clearcost
  data-origin="US" data-dest="DE"
  data-price="120" data-currency="USD"
  data-l="20" data-w="15" data-h="10"
  data-weight="1.2"
  data-category-key="general"
  data-mode="air"></div>

2. Custom themes via CSS variables

Theme the widget with CSS custom properties. Wrap the widget in a container with your --cc-* variable overrides.

Dark theme

Brand theme

.dark-theme {
  --cc-bg: #1f2937;
  --cc-border: #374151;
  --cc-text: #f9fafb;
  --cc-text-secondary: #d1d5db;
  --cc-btn-bg: #4f46e5;
  --cc-btn-border: #4f46e5;
  --cc-btn-text: #ffffff;
}

3. Available CSS custom properties

Variable Default Description
--cc-font-family system-ui Font stack
--cc-bg #ffffff Background color
--cc-border #e5e7eb Border color
--cc-radius 12px Border radius
--cc-shadow 0 1px 2px rgba(…) Box shadow
--cc-text #111827 Primary text color
--cc-text-secondary #374151 Row label color
--cc-text-muted #6b7280 Incoterm / muted text
--cc-btn-bg #ffffff Button background
--cc-btn-border #e5e7eb Button border
--cc-btn-radius 8px Button border radius
--cc-btn-text inherit Button text color

4. Integration code

Script tag (vanilla):

<script src="https://cdn.jsdelivr.net/npm/@clearcost/widget"></script>
<script>
  ClearCostWidget.mountAll({
    proxyUrl: '/api/clearcost/quote'
  });
</script>

React:

import { ClearCostQuote } from '@clearcost/widget/react';

<ClearCostQuote
  origin="US" dest="DE"
  price={120} currency="USD"
  l={20} w={15} h={10}
  weight={1.2}
  proxyUrl="/api/clearcost/quote"
/>

Vue:

import { ClearCostQuote } from '@clearcost/widget/vue';

<ClearCostQuote
  origin="US" dest="DE"
  :price="120" currency="USD"
  :l="20" :w="15" :h="10"
  :weight="1.2"
  proxy-url="/api/clearcost/quote"
/>
