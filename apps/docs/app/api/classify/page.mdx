# Classify API

**Endpoint**: `POST /v1/classify`
**Auth**: API key (`Authorization: Bearer ck_...`)
**Idempotency**: Send `Idempotency-Key`.

```bash
curl -X POST "$API_BASE/v1/classify" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Idempotency-Key: $IDEM" \
  -H "Content-Type: application/json" \
  -d '{"title":"Vintage violin from 1920, spruce top"}'
